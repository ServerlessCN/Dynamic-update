## 产品背景：

API 网关之前的控制台概览页较为单一，展示的信息较少，告警等全局配置项缺乏入口。

## 产品优化：

从单栏式的概览页改为双栏式，新增快速入口、异常告警、配额限制、最新公告等四个特色模块，提升用户体验

## 发布时间：

2020-08-06
